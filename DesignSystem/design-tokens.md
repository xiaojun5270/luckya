# UI PRO 设计令牌

## 风格选择
- 风格：Clean Tech + Glassmorphism
- 目标：科技感、专业、清晰、适合控制台与仪表盘

## 色板
- Primary: `#2563EB`
- Secondary: `#22D3EE`
- Accent: `#F59E0B`
- Background: `#0B1220`
- Surface: `#111827`
- Card: `rgba(255,255,255,0.08)`
- Border: `rgba(255,255,255,0.12)`
- Success: `#10B981`
- Warning: `#F59E0B`
- Error: `#EF4444`
- Text Primary: `#F8FAFC`
- Text Secondary: `#94A3B8`

## 设计语言
- 圆角：16 / 20 / 24
- 卡片：半透明 + 模糊 + 细描边
- 阴影：深色柔影
- 图标：SF Symbols
- 字体：iOS 系统字体 + 大标题强调

## 页面原则
- 首页先状态，后操作
- 关键数据卡片化
- 操作入口不超过两级
- 适配深色模式优先
