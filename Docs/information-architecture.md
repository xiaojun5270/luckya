# 信息架构

## 一级导航
- Dashboard
- Services
- Activity
- Settings

## Dashboard
- 顶部欢迎区
- 连接状态
- 核心指标（在线服务数 / CPU / 内存 / 最近事件）
- 快捷操作（刷新、重启服务、打开控制台、查看日志）
- 最近活动

## Services
- 服务卡片列表
- 搜索/筛选
- 服务详情
  - 运行状态
  - 启动时间
  - 资源占用
  - 快捷操作

## Activity
- 操作日志
- 错误事件
- 告警记录

## Settings
- 服务器地址
- 账号管理
- Face ID / 生物识别
- 通知设置
- 主题设置
