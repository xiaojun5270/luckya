# Lucky Remote Panel · 产品说明（v0.1）

## 定位
一个面向自有设备/自有后台的 **远程管理面板 App**，提供登录、总览、服务状态、快捷操作、日志查看、设备信息、设置等能力。

> 目标不是做系统级隐蔽远控，而是做合规、自用、可维护的 iOS 管理面板。

## 当前勘察基线
目标站点：`http://192.168.9.102:16601`
- 存在登录页
- 已使用用户提供账号进入后台
- 适合先按“管理面板”信息架构进行 iOS 映射，再逐步补 API 对接

## MVP 模块
1. 登录
2. 首页总览 Dashboard
3. 服务列表 / 状态卡片
4. 快捷操作区
5. 日志与事件
6. 设置与连接配置

## 第二阶段
1. 实时状态刷新
2. 图表与运行指标
3. 多主机切换
4. Push 通知
5. WebSocket / SSE 实时事件
6. 生物识别解锁
